Trabalhando com listas

Listas Ordenadas

Listas Ordenadas em HTML são delimitadas com a tag OL e enumeradas pela tag LI

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Listas Não Ordenadas

Listas Não Ordenadas em HTML são delimitadas pela tag UL e enumeradas pela tag LI

  • item x
  • item y
  • item k
  • item j

Misturando as coisas

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
  3. Item 3
  4. Item 4
  5. Item 5

Listas de definições

Como se fosse um dicionário, contendo sempre os termos e as definições. Toda lista de definição é delimitada pela tag DL. Todo termo é usado a tag DT e toda definição utiliza a tag DD.

Exemplo:

HTML
Linguagem de Marcação utilizada para criar o conteúdo em sites.
CSS
Linguagem de Marcação usada para especificações de estilos em sites.
JavaScript
Linguagem de programação utilizada para criar a interatividade em sites.
